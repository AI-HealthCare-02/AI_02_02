### 확정된 핵심 규칙 (변경 시 반드시 근거 확인)
- **90분 쿨다운**: 건강질문 간 최소 90분 간격. AI 채팅만 적용 (직접입력에는 미적용)
- **4대 삽입 조건**: 90분 경과 + 유효 윈도우 + 미응답 + 비야간(22-07시)
- **묶음 최대 2개**: AI 응답 1회당 질문묶음 최대 2개. 예외 없음
- **터치포인트**: 아침07-09(묶음1+묶음2★), 점심11:30-13:30(묶음3), 저녁17-20(묶음4+5)
- **묶음2★**: A그룹 전용. 오전식사 1문항 + 복약확인 1문항 = 2문항 (채소→저녁 이동)
- **묶음5 (저녁)**: 채소 + 산책 = 2문항. 야식(Q8)은 하루마무리(20:00-21:30)로 이동
- **질문 병합 확정**: Q1+Q2(수면)→5택복합 / Q12+Q13(정서)→5택복합 / Q8+Q9는 1문항+조건부서브
- **FINDRISC 100% 커버**: 8개 변수 전부 수집. 빼면 빈 칸, 더 넣으면 근거 없이 피로도만 상승
- **하루 총 응답 시간 ~22초** (B/C그룹 8문항+서브, A그룹 9문항+서브, 격일 +2문항)

### 듀얼채널 설계 (확정)
- **AI 채팅**: 90분 쿨다운 + 최대 2묶음 + 시간윈도우 적용
- **직접입력 (사이드패널)**: 쿨다운 미적용, 시간윈도우만 적용, 1카테고리씩
- **First Answer Wins**: 어느 채널이든 먼저 응답 → 다른 채널 스킵
- **4카테고리**: 🍽️식사(아침/점심/저녁/채소/구성) / 🍭당섭취 / 🏃활동(운동+산책) / 💧수분
- **하루마무리**: 20:00-21:30 사이드패널 위젯, 야식 입력 전용, 21:30 신규 오픈 불가

### 미접속/저사용 전략 (확정)
- 5단계 참여 상태: ACTIVE(80%+) → MODERATE → LOW → DORMANT → HIBERNATING
- 복귀 시 최대 1묶음(2문항)만. 폭탄 투하/죄책감 유발 절대 금지
