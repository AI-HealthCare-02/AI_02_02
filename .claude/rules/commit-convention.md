# 커밋 메시지 컨벤션 (자동 로드)

## 형식
```
<이모지> <타입>: <간결한 요약>

<본문: 무엇을/왜 변경했는지>
```

## 타입 목록
| 이모지 | 타입 | 의미 |
|--------|------|------|
| ✨ | feat | 새로운 기능 추가 |
| 🐛 | fix | 버그 수정 |
| 💡 | chore | 기능 추가 없이 코드 수정 (오타, 주석 등) |
| 🎨 | style | 코드 포매팅 수정 |
| 📝 | docs | 문서 수정 |
| 🚚 | build | 빌드 관련 파일 수정 |
| ✅ | test | 테스트 코드 추가/변경 |
| ♻️ | refactor | 리팩터링 (기능 변화 없음) |
| 🚑 | hotfix | 긴급 수정 |

## 예시
```
✨ feat: 온보딩 설문 페이지 구현

diabetes.js 데이터 기반 단계별 질문 UI + FINDRISC 점수 계산
```

## 강제 메커니즘
- `scripts/hooks/commit-msg`: git commit 시 자동 검증
- 형식 불일치 시 커밋 차단 + 사용 가능한 타입 안내
