# 기술 평가 기준 (자동 로드)

코드 작업 시 아래 기술 기준을 자동으로 참조합니다.
검증 프로토콜(3단계)은 `/evaluate` (SKILL.md) 참조.

## 기술 기준

- API 설계: HTTP Method/URI 책임 분리 + 인증/인가 구조 문서화
- 모델 연동: 성능 지표 2개 이상 + 실험 비교 결과 보존
- 비동기 처리: 적용 전후 성능 수치 기록 필수
- P95 latency 3초 이내
- 정량 근거: 기술 선택에 수치·논문·데이터 근거 필수
- 배포 안정성: 핵심 3기능(예측·대시보드·챌린지) 동작 확인
- 로그에 민감정보(혈압·혈당·건강기록) 남기지 않기
