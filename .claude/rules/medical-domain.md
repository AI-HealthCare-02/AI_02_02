# 의료 도메인 가드레일

## 질문 추가/삭제 판단 기준
- 질문 추가: 반드시 논문 근거 필요. 근거 없는 질문 추가 불가
- 질문 삭제: 대시보드/예측/챌린지 중 빈 칸 생기면 삭제 불가
- 현재 10문항이 최소이자 충분한 설계

## 핵심 논문 근거 (간략)
| 묶음 | 근거 | 효과 |
|------|------|------|
| 수면 | Cappuccio 2010 | 6h이하 → 28%↑ |
| 운동 | DPP 2002 NEJM | 150분/주 → **58%↓** |
| 식단 | Hu 2012 BMJ | 백미 최고섭취 55%↑ |
| 복약 | Cramer 2004 | 비순응 35-65%↑ |
| 정서 | Mezuk 2008 | 우울→당뇨 60%↑ |

## Phase별 예측 정확도
- Phase 1 MVP: AUC ~0.80 (FINDRISC 수준)
- Phase 2: AUC 0.82-0.85 (KDRS 추가)
- Phase 3: AUC 0.88-0.90 (혈액검사 포함)

## 절대 금지
- 의료적 진단/처방 표현
- "~해야 합니다" 대신 "~을 권장합니다" / "전문가 상담을 추천합니다"

## AI 응답 안전장치
- AI 응답은 항상: "교육 목적 / 생활습관 참고 / 전문가 상담 대체 아님"
- 혈압·혈당·건강기록은 민감정보 — 최소 수집·최소 노출 원칙
