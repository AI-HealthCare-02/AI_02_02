# Claude Design 가이드 및 다나아 UI 스타일 정리 플랜

## 한 줄 결론

이번 라운드에서는 `Do it OS 실제 서비스 접목`을 진행하지 않고, 먼저 `Claude Design 사용법 가이드`와 `다나아 UI/프론트 스타일 정의 및 톤앤매너 정리`만 진행한다.

---

## 쉬운 뜻

지금은 새 기능을 다나아에 붙이는 개발 작업을 하지 않는다.

먼저 아래 두 가지 자료를 만든다.

1. Claude Design을 어떻게 쓰는지 알려주는 한국어 가이드
2. 다나아 웹서비스의 색, 글자, 카드, 버튼, 사이드바, 채팅 화면 같은 UI 규칙 정리

이 자료가 있어야 나중에 Do it OS나 다른 화면을 만들 때 다나아답게 일관된 디자인으로 만들 수 있다.

---

## 이번 작업에서 하지 않는 것

- Do it OS를 실제 다나아 앱에 붙이지 않는다.
- 사이드바에 Do it OS 메뉴를 추가하지 않는다.
- `/app/do-it-os` 라우트를 만들지 않는다.
- DB, API, 백엔드 변경을 하지 않는다.
- AI 채팅 답변 아래 Do it OS 저장 카드를 붙이지 않는다.
- localStorage 기반 생각넣기 기능을 실제 서비스 코드에 구현하지 않는다.

---

## 이번 작업에서 하는 것

### Task A. Claude Design 사용법 가이드 만들기

목표:

- Claude Design 화면을 캡처한다.
- 각 화면에서 무엇을 누르고, 무엇을 입력하고, 어떤 결과가 나오는지 한국어로 설명한다.
- 코딩을 모르는 사람도 Claude Design을 보고 따라 할 수 있게 만든다.

예상 산출물:

```text
output/claude_design_guide.html
output/claude_design_guide.md
output/claude_design_handoff.yaml
```

주의:

- 로그인 세션 파일은 절대 커밋하지 않는다.
- `output/cd_session.json` 같은 파일은 로컬 전용이다.
- 캡처가 실패하면 억지로 진행하지 않고 수동 캡처 방식으로 전환한다.

---

### Task B. 다나아 UI/프론트 스타일 정의 및 톤앤매너 정리

목표:

- 다나아 웹서비스가 어떤 색, 여백, 둥근 모서리, 카드 스타일, 텍스트 위계, 채팅 말풍선, 사이드바 구조를 쓰는지 정리한다.
- 다크 모드와 라이트 모드의 색상 토큰을 명확히 정리한다.
- 앞으로 Claude Design이나 Do it OS 목업을 만들 때 참고할 수 있는 디자인 기준 문서를 만든다.

정리할 항목:

- 전체 톤앤매너
- 색상 토큰
- 라이트/다크 모드
- 타이포그래피
- 카드 스타일
- 버튼 스타일
- 사이드바 스타일
- 중앙 채팅 영역
- 오른쪽 Today 패널
- 리포트/챌린지/설정 화면 공통 규칙
- 피해야 할 스타일
- Claude Design에 넣을 예시 프롬프트

예상 산출물:

```text
docs/setup/MM-DD HH-mm 다나아 UI 프론트 스타일 정의.md
또는
Do it OS/MM-DD HH-mm 다나아 UI 프론트 스타일 정의.md
```

---

## 권장 작업 순서

1. 현재 다나아 화면과 코드에서 실제 UI 토큰을 확인한다.
2. 다크/라이트 색상과 RGB/HEX 값을 정리한다.
3. 채팅, 사이드바, 오른쪽 패널, 리포트, 챌린지, 설정의 공통 스타일을 묶는다.
4. Claude Design에서 사용할 수 있는 프롬프트 예시를 만든다.
5. Claude Design 캡처가 가능한지 확인한다.
6. 캡처가 가능하면 HTML 가이드를 만든다.
7. 캡처가 막히면 수동 캡처 기준으로 가이드를 만든다.

---

## 안전 원칙

- 이 작업은 문서/가이드 작업이다.
- 제품 코드 변경은 하지 않는다.
- 로그인 세션, 쿠키, 토큰은 커밋하지 않는다.
- output 폴더는 결과 확인용으로만 쓴다.
- 실제 서비스에 Do it OS를 붙이는 작업은 별도 플랜과 별도 브랜치에서 진행한다.

---

## 나중에 이어서 할 작업

Do it OS를 다나아 웹서비스에 실제로 붙이는 작업은 아래 플랜으로 분리한다.

```text
Do it OS/Do it OS 다나아 통합 MVP 플랜.md
```

이 플랜은 아직 실행하지 않는다.

먼저 Claude Design 가이드와 다나아 UI 스타일 정의가 완성된 뒤 다시 검토한다.
