📖 각 영역 명칭 & 설명

①
사이드바
Sidebar / Navigation Bar / Side Nav
화면 왼쪽에 고정된 세로 메뉴 영역입니다. 앱의 주요 페이지(탭)를 이동하는 내비게이션 역할을 합니다. 위쪽에 질환 카테고리, 아래쪽에 기능 메뉴가 배치됩니다.
다른 이름: 좌측 네비게이션, 드로어(Drawer), 사이드 네비(Side Nav), 앱 쉘(App Shell)의 일부
②
대시보드 스트립
Dashboard Strip / Top Bar / Stats Bar / KPI Strip
메인 화면 최상단에 위치하는 얇은 가로 띠 영역입니다. 핵심 건강 지표(위험도, 건강 응답 수, 수면 시간)를 한눈에 보여주는 역할입니다. 어떤 탭을 보든 항상 보이는 글로벌 상태 요약입니다.
다른 이름: 톱 바(Top Bar), 상단 통계 바, 헤더 스트립, KPI 대시보드
③
메인 콘텐츠 영역
Main Content Area / Chat Area
화면의 가장 큰 중앙 부분으로, 현재 탭의 주요 콘텐츠가 표시되는 곳입니다. "다나아 AI" 탭에서는 AI 채팅 인터페이스가, "AI 건강 리포트" 탭에서는 건강 분석 서브탭이 표시됩니다.
다른 이름: 콘텐츠 에어리어, 메인 패널, 바디(Body), 워크스페이스(Workspace)
④
사이드 패널
Side Panel / Widget Panel / Right Sidebar / Aside
메인 콘텐츠 오른쪽에 위치하는 보조 정보 영역입니다. 채팅의 핵심 흐름을 방해하지 않으면서 부가 정보(건강운세, 날씨, 오늘 통계, 바로가기 등)를 제공합니다.
다른 이름: 위젯 패널, 우측 사이드바, 디테일 패널(Detail Panel), 인포 패널(Info Panel)
⑤
유저 프로필
User Profile / Account Badge
사이드바 하단에 위치하는 현재 로그인한 사용자의 정보 표시 영역입니다. 아바타(프로필 이미지), 이름, 분류 그룹 등을 표시합니다.
다른 이름: 어카운트 뱃지, 프로필 카드, 유저 아바타
⑥
인라인 위젯 카드
Inline Widget / Rich Card / Embedded Card
AI 봇 메시지 안에 포함되는 구조화된 정보 카드입니다. "이번 주 건강 리포트" 같은 표(테이블)형 데이터를 채팅 버블 내에서 깔끔하게 보여줍니다.
다른 이름: 리치 메시지(Rich Message), 카드 메시지, 구조화된 응답(Structured Response)
⑦
건강 질문 카드
Health Question Card / Interactive Card / Survey Card
AI가 사용자에게 건강 관련 질문을 던지고 버튼으로 답변을 받는 인터랙티브 카드입니다. "수면 충분하셨나요?", "아침 드셨나요?" 같은 질문 + 선택지 버튼 구성입니다.
다른 이름: 인터랙티브 카드, 설문 카드, 퀵 리플라이(Quick Reply), 선택형 응답
⑧
트리거 카드
Trigger Card / Quick Action / Floating Prompt / Contextual Card
입력란 바로 위에 뜨는 컨텍스트 기반 액션 카드입니다. 시간대에 따라 "점심 드셨어요?" 같은 체크인 프롬프트가 나타나고, 물 기록·스트레칭 같은 퀵 액션 버튼을 제공합니다. ChatGPT의 입력창 위 제안 카드와 유사한 패턴입니다.
다른 이름: 플로팅 프롬프트, 컨텍스트 카드, 액션 시트(Action Sheet), 서제스천 칩(Suggestion Chip)
⑨
메시지 입력란
Input Bar / Message Composer / Chat Input / Prompt Bar
화면 최하단에 고정된 텍스트 입력 영역입니다. 사용자가 AI에게 메시지를 보내는 곳으로, 체크인 버튼과 전송 버튼이 함께 있습니다.
다른 이름: 프롬프트 바, 컴포저(Composer), 인풋 필드, 채팅 바(Chat Bar)
+
채팅 버블
Chat Bubble / Message Bubble
각 메시지의 둥근 말풍선 모양 컨테이너입니다. AI(왼쪽)와 사용자(오른쪽)의 메시지가 서로 다른 스타일로 표시됩니다. AI 버블 안에 위젯 카드나 건강 질문 카드가 포함될 수 있습니다.
다른 이름: 말풍선, 메시지 카드, 메시지 컨테이너

📏 현재 레이아웃 비율

사이드바 180px
채팅 메인 (flex:1)
사이드 패널 280px
• 사이드바 (180px) — 고정 너비. 네비게이션 메뉴
• 채팅 메인 (flex:1) — 남은 공간을 모두 차지. 채팅 메시지 + 트리거 카드 + 입력란
• 사이드 패널 (280px) — 고정 너비. 건강운세/날씨/통계 등 위젯

예시: 화면 너비가 1400px일 때 → 사이드바 180 + 채팅 940 + 사이드패널 280 = 1400px
