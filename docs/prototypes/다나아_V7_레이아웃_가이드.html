2. 컴포넌트 카탈로그

V7에서 사용되는 주요 UI 컴포넌트별 클래스명, 역할, 참고사항입니다.

① 사이드바 Layout

.sidebar

왼쪽 고정 네비게이션. 카테고리(당뇨/고혈압 등) + 메뉴(다나아AI/건강리포트/챌린지/설정) + 대화 히스토리 + 미션 + 프로필로 구성.

너비: 260px / margin-right: 8px (메인과 간격)

② 대화 히스토리 Widget

.sidebar-history

이전 대화 세션 목록. "새 대화" 버튼 + 날짜별 그룹(오늘/어제) + 대화 항목. 클로드AI 사이드바 참고.

위치: .sidebar 내 메뉴 섹션 아래, .sidebar-bottom 위

③ 오늘의 미션 Widget

.sidebar-mission

오늘 목표 진행률 표시. 프로그레스 바 + 퍼센트 텍스트. 그라데이션 바(primary→accent).

위치: .sidebar-bottom 내부, 프로필 위

④ 대시보드 스트립 Interactive

.dash-strip-wrap → .dash-strip

상단 요약 카드 3장(혈당/BMI/FINDRISC). ▲▼ 토글로 접기/펼치기. CSS transition 애니메이션.

JS: toggleDashStrip() / 접힌 상태: .dash-strip.collapsed

⑤ 채팅 메시지 영역 Core

.chat-messages

AI 대화 인터페이스. 사용자/AI 버블 + 인라인 건강 질문 위젯(.chat-widget). 스크롤바 오른쪽 끝 고정.

padding: 24px 24px / 스크롤바: 5px 너비, 투명 트랙

⑥ LIVE 넛지 바 New in V7

.live-nudge-bar

입력란 위 실시간 알림. LIVE 빨간 배지 + 2개 메시지 + 닫기(✕). 슬라이드업 애니메이션.

JS: dismissLiveNudge() / max-width: 720px 중앙 정렬

⑦ 채팅 입력란 Core

.chat-input-area → .chat-input-wrap

메시지 입력 + 전송(↑) 버튼. 둥근 pill 형태(border-radius:24px). V7에서 체크인 버튼 제거.

.chat-input-wrap: max-width:720px, margin:0 auto

⑧ 사이드 패널 Interactive

.chat-side-panel

오른쪽 정보 패널. 접기/펼치기 가능. 펼침: 280px(위젯 표시). 접힘: 48px(아이콘만).

JS: toggleSidePanel() / 접힌 상태: .chat-side-panel.collapsed
토글버튼: .side-panel-toggle / 아이콘: .side-panel-collapsed-icons

⑨ 오늘 한눈에 Widget

.side-widget → .mini-stats

식사(2/3끼), 수분(5/8잔), 운동(10/30분), 수면(5.5h) 프로그레스바 4줄. 컬러별 구분.

위치: 사이드패널 내 첫 번째 위젯

⑩ 건강운세 플립카드 Interactive

.fortune-flip-wrap → .fortune-flip-inner

3D CSS 플립 카드. 뒷면(그라데이션+🍀), 앞면(운세+이번주 7일 그리드). 클릭 토글.

JS: toggleFortune() / perspective: 800px / rotateY(180deg)
