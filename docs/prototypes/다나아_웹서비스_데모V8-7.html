우리 프로젝트에서 FastAPI랑 Redis Stream 연결하는 부분 어떻게 구현하면 돼? Producer-Consumer 패턴이라고 했는데
오전 7:32
다
FastAPI에서 Redis Stream 연결은 이렇게 구현해요:

① Producer (FastAPI)
redis.xadd("stream:ai", {"task": data})
→ 사용자 요청을 Stream에 push

② Consumer (AI Worker)
redis.xreadgroup("group", "worker1", ...)
→ Consumer Group으로 분산 처리

③ ACK 처리
redis.xack("stream:ai", "group", msg_id)
→ 처리 완료 후 확인 응답

설정은 app/core/redis.py에서 관리하고 있어요.
