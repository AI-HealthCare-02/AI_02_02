01 EVIDENCE-BASED DESIGN

설계 근거

국제적으로 검증된 당뇨 위험도 평가 도구 3가지를 크로스 분석하여 최적의 질문 세트를 도출했습니다.

FINDRISC — 핀란드

Finnish Diabetes Risk Score

8문항, 비침습적 글로벌 표준 도구

  • 나이, BMI, 허리둘레, 혈압약
  • 고혈당 이력 (단독 5점, 최대 배점)
  • 운동량, 채소/과일 섭취
ADA Risk Test — 미국

American Diabetes Association

7문항, 60초 완료 목표

  • 나이, 성별, 임신성당뇨, 가족력
  • BMI ≥30 = 최대 3점
  • 고혈압, 운동량
Korean Score — 한국

한국형 당뇨 위험도 점수

6문항, KNHANES 기반 한국인 특화

  • 나이 (45세+ = 3점)
  • 허리둘레 (한국인 기준 적용)
  • 흡연 1점, 음주 최대 3점
💡

핵심 발견

나이 + BMI/허리둘레 + 가족력 + 운동량 — 이 4가지만으로 당뇨 위험 예측력의 약 80%를 확보할 수 있다. 나머지는 정확도를 높이는 보조 변수.

05 ANTI-DROPOUT UX

이탈 방지 UX 전략

설문 완료율을 80% 이상으로 유지하기 위한 7가지 전략입니다.

📊

프로그레스 바 + 남은 시간

화면 상단에 얇은 진행 바와 "약 2분이면 끝나요" 표시. 끝이 보여야 사용자가 완주합니다.

💬

"왜 물어보는지" 한 줄 설명

매 화면마다 "이 질문이 왜 중요한지" 보조 텍스트를 제공합니다. 이유를 알면 답변 동기가 생깁니다.

⏭️

"모르겠어요" 항상 제공

모든 질문에 탈출구를 둡니다. 수치 입력처럼 어려운 질문은 "모름"이 기본 선택으로 하이라이트.

🎉

중간 보상 메시지

Phase 전환 시 "기본 프로필 완성!", "거의 다 됐어요!" 등 격려 메시지로 완주 동기를 유지합니다.

☝️

1화면 = 1질문 원칙

인지 부하를 최소화. 예외적으로 Screen 10만 3개를 묶지만, 모두 "하나만 고르면 되는" 간단한 항목.

👆

클릭 전용 — 키보드 0회

모든 답변은 버튼 탭, 슬라이더 드래그, 토글 칩. 키보드가 올라오는 순간이 없어 모바일 UX를 극대화.

💾

이탈 시 자동 저장 + 복귀 유도

중간에 나가도 답변한 데이터는 자동 저장. 재접속 시 "이어서 할까요?" 팝업을 표시하고, 미완료 상태에서도 수집된 데이터만으로 기본 대시보드를 제공합니다.
