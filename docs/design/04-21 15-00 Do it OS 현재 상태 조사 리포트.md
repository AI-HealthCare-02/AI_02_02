## 1. Do it OS 개념 정리

### 1-1. 한 문장 정의

> 사용자가 머릿속 생각 · 할 일 · 일정 · 프로젝트 · 걱정 · 건강 단서를 **빠르게 꺼내고**, 사용자가 **확인한 것만 분류해서 저장**하도록 돕는 생활 관리 레이어.

### 1-2. 이론 배경 (문의 프롬프트 §70~98 요약)

| 개념 | 다나아 재해석 |
|---|---|
| 브레인덤프 | "생각넣기 / 생각쏟기" 입력창 — 떠오르는 대로 쏟아내기 |
| 자이가르닉 효과 | 미완료 생각을 캔버스에 꺼내 두면 머릿속 반복 부담 ↓ |
| 수면 전 인지 각성 | 자기 전 생각을 밖으로 옮겨 잠들기 편하게 |
| GTD | `생각넣기 → 분류 확인 → 확인 후 저장` 흐름 |
| PARA | 원본 그대로가 아니라 **프로젝트 / 일정 / 노트 / 건강 단서** 로 축소 |
| Second Brain | 일상 맥락 + 건강 단서를 함께 보관 |

### 1-3. 절대 지킬 제품 방향 (문의 프롬프트 §103~113, 1차플랜 §1)

- 다나아의 중심 UX(AI 채팅) 유지. 기존 3대 메뉴(AI 채팅 · 리포트 · 챌린지) 구조 불변.
- 오른쪽 Today 패널 = 건강 기록 입력 그대로.
- Do it OS 는 **건강 기록을 대체하지 않는다**.
- AI 자동 저장 금지. 저장 후보만 제시 → 사용자 수동 확인 시 저장.
- 의료 진단·처방·약물 안내로 확장 금지.
