> **문서 버전**: v1.0

> **대상 카테고리**: 당뇨 (1차 오픈), 고혈압·비만·심혈관질환·기타 (추후 확장)

---

## 1. 서비스 개요

### 1.1 서비스 정의

다나아는 AI 채팅 인터페이스에 짧은 건강 질문을 자연스럽게 삽입하여, 사용자의 생활습관 데이터를 수동적으로 수집하고, 이를 기반으로 만성질환(당뇨) 위험도를 예측·추적하는 웹서비스이다.

### 1.2 핵심 가치

- **무의식적 데이터 수집**: 별도 기록 앱 없이, AI 대화 중 자연스럽게 건강 데이터 축적
- **근거 기반 위험 예측**: FINDRISC + 한국형 당뇨 위험도 점수(KNHANES 기반) 하이브리드 모델
- **행동 유도형 개인화**: 수집된 데이터 기반 맞춤 챌린지·리포트 제공

### 1.3 기술 스택 (계획)

| 항목 | 기술 |
|------|------|
| 프레임워크 | Next.js (App Router) |
| 프론트엔드 | React, Tailwind CSS |
| 백엔드 API | Next.js API Routes |
| 데이터베이스 | 추후 확정 (Supabase / Firebase 등) |
| AI 엔진 | 추후 확정 (OpenAI / Claude API 등) |
| 배포 | Vercel |
