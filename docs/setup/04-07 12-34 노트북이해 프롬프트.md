# 노트북이해 프롬프트

이 문서는 데스크탑에서 작업하던 DANAA 프로젝트를 노트북에서 거의 같은 상태로 이어서 작업하기 위한 현재 상태 요약이다.
노트북에서 Claude Code 또는 다른 AI 코딩 도구를 열었을 때, 먼저 이 문서를 읽히고 시작한다.

## 6. Superpowers / GStack 상태

이번 노트북 이동본에는 프로젝트만이 아니라 Claude 도구 복원용 자료도 같이 넣었다.

포함 대상:

- Superpowers 캐시 복사본
- GStack 로컬 복사본
- GStack/Superpowers 예시 설정 파일

중요:

- `Superpowers`는 현재 데스크탑에서 플러그인 캐시에 설치된 상태다.
- `GStack`은 현재 프로젝트 저장소 안이 아니라 별도 작업 복사본에 있다.
- 그래서 이전 프로젝트 zip만 풀어서는 이 둘이 자동 복원되지 않았다.
- 이번 묶음에는 노트북에서 복원할 수 있게 관련 폴더를 같이 넣는다.

## 8. 노트북에서 Claude 도구 복원

압축 해제 후 아래 폴더를 기준으로 복원한다.

- `claude-home/plugins/cache/claude-plugins-official/superpowers/5.0.7`
- `claude-home/skills/gstack`
- `claude-home/settings.superpowers-example.json`

### 8-1. GStack 복원

노트북 PowerShell에서:

```powershell
New-Item -ItemType Directory -Force "$env:USERPROFILE\\.claude\\skills" | Out-Null
Copy-Item -Path ".\\claude-home\\skills\\gstack" -Destination "$env:USERPROFILE\\.claude\\skills\\gstack" -Recurse -Force
```

### 8-2. Superpowers 복원

```powershell
New-Item -ItemType Directory -Force "$env:USERPROFILE\\.claude\\plugins\\cache\\claude-plugins-official\\superpowers" | Out-Null
Copy-Item -Path ".\\claude-home\\plugins\\cache\\claude-plugins-official\\superpowers\\5.0.7" -Destination "$env:USERPROFILE\\.claude\\plugins\\cache\\claude-plugins-official\\superpowers\\5.0.7" -Recurse -Force
```

### 8-3. 그래도 인식이 안 되면

Claude Code에서 다시 설치:

```text
/plugin install superpowers@claude-plugins-official
```

그리고 Claude Code를 재시작한다.

### 8-4. GStack 참고

- GStack은 `Node.js`와 `Bun`이 있어야 안정적이다.
- 노트북에 `Git`, `Node.js`, `Bun`, `Claude Code`가 먼저 있어야 한다.
- 복원 후 Claude Code 재시작 뒤 `/gstack-*` 명령이 보이는지 확인한다.

## 9. 프로젝트 환경 복구

이번 묶음은 휴대성을 위해 아래 폴더는 제외하고 만들 수 있다.

- `.venv`
- `frontend/node_modules`
- `.pytest_cache`
- `.ruff_cache`

따라서 노트북에서 아래 복구가 필요하다.

### Python

```powershell
uv sync --group app --group dev
```

### Frontend

```powershell
npm --prefix frontend install
```

### Docker가 필요한 경우

```powershell
docker compose -p danaa-dev up -d postgres redis fastapi
docker compose -p danaa-dev exec fastapi uv run python -m aerich upgrade
```
