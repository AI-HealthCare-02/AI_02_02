## A. 아키텍처 / 경계 준수 게이트
- [x] 기존 모듈 경계와 소유권을 깨지 않았다.
- [x] 임시 우회 / 단기 패치 / 레이어 침범이 없다.
- [x] 캐시 / revalidate / selector 책임이 정리되어 있다.

리뷰 근거:
- 테마 상태 관리는 `ThemeContext` / `useTheme`로 분리
- 페이지별 렌더링 책임은 유지
- 백엔드 설정 계약은 기존 `/api/v1/settings`에 확장만 적용
- `ThemeContext`의 잘못된 훅 위치를 정상 `useEffect` 구조로 수정
