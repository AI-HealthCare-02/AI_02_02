# 클로드코드 전달 가이드

## 먼저 알려줘야 할 핵심
Claude Code에게는 아래 네 가지만 먼저 알려주면 됩니다.

1. 실제 프론트 수정 대상은 `frontend`
2. Python 스크립트 작업 대상은 `tools/python`
3. 참고 자료는 `docs/planning`, `docs/prototypes`
4. `archive` 아래는 보관용이므로 건드리지 말 것

## 시작 프롬프트
```text
현재 작업 루트는 C:\Users\mal03\Desktop\레퍼런스\마지막 웹프로젝트 입니다.

실제 프론트 개발 코드는 frontend 에 있습니다.
Python 스크립트나 보조 작업은 tools/python 에서만 진행해주세요.
docs/planning 과 docs/prototypes 는 참고 자료입니다.
archive 아래 폴더들은 이전 작업물 보관용이므로 수정하지 말아주세요.

먼저 현재 frontend 와 tools/python 구조를 분석하고,
참고 자료 중 어떤 문서가 현재 작업과 직접 관련 있는지 추려서 설명해주세요.
설명 후에 제가 승인하면 구현을 진행해주세요.
```

## 이전 작업물을 전달하는 방법
이전 Claude Code 작업물이 필요하면 경로를 직접 지정해서 알려주면 됩니다.

예시:
```text
참고할 이전 작업물은 archive/ai-work/claude웹프로젝트 아래에 있습니다.
필요하면 해당 폴더 내용을 읽고 현재 frontend 에 반영 가능한 부분만 정리해주세요.
기존 구조를 그대로 복사하지 말고, 현재 Next.js 구조에 맞게 해석해주세요.
```

## 가장 안전한 작업 습관
1. Claude Code는 항상 루트에서 시작
2. 프론트 구현은 항상 `frontend`
3. Python 스크립트는 항상 `tools/python`
4. 문서 참고는 항상 `docs/...`
5. 보관자료는 항상 `archive/...`
6. 새 세션을 열면 이 문서의 시작 프롬프트를 먼저 전달
